Способ задания графа:
Создаём булевый нулевой граф, считываем смежные рёбра из файла и обозначаем их в графе единицой.

Переменные:
int countVertex - количество вершин
int countEdge - количество рёбер
string symbol - переменная для считывания символа из файла
bool **graph - сам граф
int first - первая вершина ребра
int second - вторая вершина ребра
bool *visited - массив посещённых вершин
int mainCount - изначальное количество компонентов графа
bool **deletedVertex - массив вершин, удаление которых приведёт к увеличению компонентов графа
bool find - проверяем наличие таких вершин

Функции:
void findSmallGraph - функция для нахождения графа из начальной точки. Также помечает посещённые вершины.
int countSmallGraph - функция для нахождения компонентов графа.
void findDeletedVertex - функция для нахождения вершин удаление которых приведёт к увелечению компонентов графа.
